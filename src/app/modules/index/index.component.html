คำถามที่พบบ่อย

Buddy Recruit คือ อะไร
เราคือ ระบบ ผู้ช่วยอัจฉริยะ ที่ช่วยจัดการใบสมัคร คัดกรอง ให้งาน Recruitment ของคุณ รวดเร็ว และ แม่นยำมากขึ้น ด้วย Feature การคัดแยกอีเมลอัตโนมัติ, อ่าน CV และ ให้คะแนนได้ตามคุณสมบัติที่เรากำหนดได้เอง ช่วยให้ไม่พลาด Talent ที่คุณต้องการ ส่งอีเมลนัดหมายได้อัตโนมัติ ทำให้ Recruitment Process ไม่มีสะดุด
Value ที่คุณจะได้รับจากการนำ Buddy Recruit ไปใช้ในองค์กร
  • ตัดปัญหาคัดแยกอีเมลไม่ทัน, ส่งข้อมูลผู้สมัครให้ Hiring Manager ล่าช้า เนื่องจาก Buddy Recruit จะจัดการคัดแยกอีเมลใบสมัครให้คุณโดยอัตโนมัติ
  • ไม่ต้องอ่านและ match CV กับ JD (Job Description) ที่ซับซ้อนอีกต่อไป เนื่องจาก Buddy Recruit ช่วย ให้คะแนน CV ตาม Qualification ที่คุณกำหนดได้เอง ทำให้ประหยัดเวลาของ Hiring Manager ไม่ต้อง Screen CV ทุกใบอีกต่อไป
  • รองรับ CV ได้หลากหลาย ทั้ง PDF, WORD, JPG
  • ลดระยะเวลา Recruitment Process ทำให้สามารถประหยัดต้นทุน Job board ได้
  • มีการจัดเก็บข้อมูลการทำ Recruitment ทำให้สามารถนำข้อมูลมาใช้ในการตัดสินใจได้ เช่น มีประวัติการสมัครงานของผู้สมัคร, มีประวัติ Blacklist, จัดเก็บ Leadtime ในการรับสมัครงานแต่ละตำแหน่ง, อื่นๆ
สมัครใช้งานยังไง
เพียงกรอกข้อมูลเบื้องต้น เพื่อสอบถามลักษณะการใช้งานและแนะนำโปรแกรมที่เหมาะสม จากนั้น Setup ระบบเพียง 1 วัน รับ User ไปใช้งานได้เลย ไม่ต้องมีค่าใช้จ่ายในการซื้อเครื่อง Server และดูแล ข้อมูลของของลูกค้า จะถูกจัดเก็บอยู่ในระบบ Cloud Platform ที่มีความปลอดภัยและน่าเชื่อถือระดับโลก
